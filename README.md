# CoinPulse

A cryptocurrency dashboard built with React, TypeScript, and Redux Toolkit. Track real-time prices, view interactive charts, and get AI-powered investment recommendations.

**Live Site:** [https://coinpulse.web.app](https://coinpulse.web.app)  
**GitHub:** [https://github.com/ahmadabdelqadir/CoinPulse](https://github.com/ahmadabdelqadir/CoinPulse)

---

## Features

- Browse and search 100+ cryptocurrencies
- Real-time price updates with live polling
- Interactive candlestick charts (1m to 30D timeframes)
- AI buy/sell recommendations powered by OpenAI
- Select up to 5 coins to track; choices persist in localStorage
- Responsive UI with parallax background and smooth animations

## Tech Stack

React · TypeScript · Vite · Redux Toolkit · React Router · TailwindCSS · Framer Motion · Recharts · CoinGecko API · CryptoCompare API · OpenAI SDK
